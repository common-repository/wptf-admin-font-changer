== Description ==
WPTF Admin Font Changer basically allows you to change the font style of the admin for a more readable font. It's lightweight, not too much code and css for flexible use. 

== Installation ==
1. Plugins > Add New > Search `WPTF Admin Font Changer`
2. Install Now > Activate


**Important**

Always try at your local website first.
